## Elemetos

## Animes

Dentro contendra anime

## Anime

Anime serie televisiva de origen japones, se relaciona con los personajes a traves de un ID

## ID

Para poder relacionarlo mas tarde con los personajes


## Título

Texto donde pondremos el titulo del anime

## Genero

Texto donde pondremos el genero del anime

## AñoEmision

Año en el que empezo a emitirse el anime

## Episodios

Número de capítulos que tiene el anime

## Imagen

Imagen del anime o de los perrsonajes, este campo es opcional tal y como se indica en el dtd

## Personajes

Contendra personaje

## Personaje

Elemento que contiene un ID, un nombre, un rol, edad e imagen del personaje

## Nombre

Nombre del personaje

## Rol

Papel que desempeña en la obra

## Edad

Edad del personaje

## Imagen

Imagen del personaje, completamente opcional


## Relaciones

## ID

El elemento ID se relaciona con el ID del personaje, de esta manera se sabe a que anime pertenece cada personaje
